## 💻 Sobre o projeto

<!-- EXPLICA O MOTIVO DO PROJETO -->
Scale Fintech é um projeto desenvolvido para praticar conceitos de HTML e CSS na criação de uma landing page voltada para o setor financeiro.

O objetivo do projeto foi simular uma fintech de controle financeiro, aplicando boas práticas de semântica, estruturação de conteúdo e estilização de interfaces modernas, com foco em organização, clareza e experiência do usuário.
